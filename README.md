# House-Price-Prediction

Deskripsi Project
Project ini bertujuan untuk membangun model machine learning dalam memprediksi harga rumah berdasarkan berbagai fitur seperti luas bangunan, jumlah kamar, dan lokasi.

Tujuan
- Memprediksi harga rumah secara akurat
- Mengidentifikasi faktor yang mempengaruhi harga rumah

Tools dan Teknologi
- Python
- Pandas
- Scikit-learn

Proses
1. Data preprocessing
2. Feature selection
3. Model training
4. Evaluasi model

Hasil
- Model mampu memprediksi harga rumah dengan cukup baik
- Faktor yang paling berpengaruh adalah lokasi dan luas bangunan

Kesimpulan
Machine learning dapat digunakan untuk membantu estimasi harga properti dan mendukung pengambilan keputusan dalam sektor real estate.
